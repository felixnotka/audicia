# Code of Conduct

Audicia follows the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing **info@audicia.io**.

All reports will be reviewed and investigated promptly and fairly. The project maintainers are obligated
to maintain confidentiality with regard to the reporter of an incident.
