## Reading This Report

- **`observedRules`** — The 4 distinct API access patterns this ServiceAccount
  uses. Each includes timestamps and call counts.
- **`suggestedPolicy`** — A ready-to-apply Role and RoleBinding granting exactly
  the observed permissions.
- **`compliance.score: 50`** — This SA uses only 4 of its 8 effective
  permissions (Yellow). The 4 unused grants include access to `secrets`.
- **`eventsProcessed: 24458`** — Total audit events that contributed to this
  report.
